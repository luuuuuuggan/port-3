Project #3 Face Generator

Introduction

In this assignment, we create a face generator using customized variables and the random() function. Plug the maximum / minimum values into the random() for each feature. The face generator creates multiple variations of the face through mouse or keyboard interactions.

Reflection

This project is really entertaining, and to interact with the face we created is extremely satisfying. I spent a relatively long time figuring out how to make the triangles(the eys) move following the rectangle (the face). And I finally figured it out by some tests. When I look at the outcome I feel it's totally worth it.
